Exploring Processing

This self portrait and processing sketch was made in processing. The self portrait is composed entirely of ellipses barring one rectangle for the neck. The hair line was achieved through using elipses as masks to seemingly create a curving line. The processing animation contains a bouncing ellipse using velocity and gravity variables that changes color on mouse press. This page also uses a p5 sketch that generates a vector point that follows your cursor at a set speed to simulate smoothness and the zig zag effect at lower speeds.
